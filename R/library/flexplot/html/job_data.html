job_data {flexplot}R Documentation

Job performance dataset

Description

These simulated data show relationships between various predictors of job performance with missing data

Usage

job_data

Format

A data frame with 187 rows and 4 variables:

job.performance

supervisor ratings of job performance

iq

intelligence

conscientiousness

score on a conscientious scale

interview

Rating on an interview scale
