secret_agent {flexplot}R Documentation

Secret agent dataset

Description

Simulated data about various characteristics of secret agents

Usage

secret_agent

Format

A data frame with xx rows and 7 variables:

muscle

bicep circumference

height

height in inches

tortured

have they been tortured on a mission?

missions

number of missions completed

Course.Score

score on an obstacle course

Branch

Branch of service (CIA versus MI6)

Salary

Annual salary
