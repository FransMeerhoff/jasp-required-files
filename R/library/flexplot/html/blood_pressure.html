blood_pressure {flexplot}R Documentation

Dataset examines what affects blood pressure.

Description

This dataset looks at how various lifestyle choices affect blood pressure.

Usage

blood_pressure

Format

A data frame with xx rows and 4 variables:

BP

Blood pressure level

Exercise

description

Vegetarian

Whether the subject is vegetarian or not.

Stress

Level of stress
