flexplot_jasp2 {flexplot}R Documentation

Flexplot in JASP

Description

This function was developed for use in JASP. It takes a dataset as input with user options and returns a flexplot graphic

Usage

flexplot_jasp2(jaspResults, dataset, options)

Arguments

jaspResults

A JASP object

dataset

dataset supplied by JASP

options

a list of options to pass to JASP

Value

a flexplot graphic.
