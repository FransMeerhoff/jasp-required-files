mixedmod_jasp {flexplot}R Documentation

General Linear Model in JASP

Description

This function was developed for use in JASP. It takes a dataset as input with user options and returns results (tables, plots, etc)

Usage

mixedmod_jasp(jaspResults, dataset, options)

Arguments

jaspResults

A JASP object

dataset

dataset supplied by JASP

options

a list of options to pass to JASP

Value

a table, plot, etc.
